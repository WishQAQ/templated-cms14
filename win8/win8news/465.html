Win8.1 2014 Update汇总：关机按钮归来
时间：2014-01-29 15:38 来源：未知

Win8.1 2014 Update成为时下的焦点，新系统的多个特性：开始屏幕加上电源按钮、更直观的搜索标识、任务栏固定Modern应用、全新的磁贴右键管理等功能，这些新功能都透露着，微软逐渐完善细节，消除传统桌面与Modern开始屏幕之间的隔阂。

Windows 8.1 2014 Update会是一个新系统吗？不是；是一个SP服务包？也不是；何时发布？3月11日；为什么选这天？刚好是微软补丁日。

相反，Win8.1 2014 Update更像是微软周二补丁日推送的功能更新补丁，但这次是一枚神奇的功能更细（包含多个新功能加入）。

▲Windows 8.1 GA Rollup A更新

微软每个月为Windows推送各类安全更新同时，也会不定期推送功能更新，主要用于小修小补。例如，2014首个补丁日，Win8.1迎来多枚功能更新，例如修复拨号651错误、改善Win8.1商店性能。

如果你是Win8/Win8.1老用户，会发现微软每个月的更新补丁体积都很大，每次都是100/200MB左右，查阅下更新详情，都是功能维护、安全维护类内容甚至是内核优化。

具体例子为，2013年10月份上市Win8.1的同时，微软也推送了Windows 8.1 GA Rollup A更新，主要修复bug和底层内核维护，从外观来看，没有任何区别。

时隔半年，Win8.1 2014 Update将对Win8.1的界面、功能带来更丰富的体验，或者可以称为“Windows 8.1 GA Rollup B”（暂时命名）更新，也许后期还会有其他的Win8.1 GA Rollup更新推送。

截至目前，Win8.1 2014 Update已知新特性：

1、免费升级，从Windows Update推送

之前，Win8用户使用Windows应用商店，下载累计4G大小的Win8.1升级包，无论服务器连接速度还是下载过程体验，都非常差劲，而这次，Win8.1 2014 Update将告别商店更新模式，直接在微软的Windows Update中心推送，因此，也是免费升级。

2、延续NT6.3内核，保证现有软件的兼容性

现有的Win8.1采用了NT6.3内核，同样Win8.1 2014 Update继续延续NT6.3内核，无论是常用软件、桌面游戏都可以继续兼容运行。

3、任务栏支持固定Modern应用

在最新的Win8.1 2014 Update开发版，原先只能固定在开始屏幕的Modern应用磁贴，这次可以固定在传统桌面的任务栏，支持实时预览，桌面用户可以更快捷切换桌面程序和Modern应用，同时也透露微软进一步融合传统桌面和Modern界面之间的隔阂感。

4、开始屏幕添加搜索标识，搜索更直观

之前，Win8.1用户使用搜索功能，需要唤出内置的超级按钮：Charms，或者快捷键Win+Q、Win+F调出；在Win8.1 2014 Update，直接在开始屏幕右上角显示搜索功能，点击，即可开启搜索功能。

5、惊现电源按钮，一键关机

刚入手Win8/Win8.1的新用户，最烦心的是传统的关机键消失了（拔电源是另回事），这次Win8.1 2014 Update开始屏幕也加入便捷的电源按钮，支持关机、重启等操作，尤其对于触控类平板电脑设备/桌面电脑用户，这个贴心功能非常到位。

6、Modern应用/桌面程序磁贴右键新功能

从Win8到Win8.1，开始屏幕Modern应用、桌面程序都是通过底部菜单栏统一管理，让桌面用户产生强烈的隔阂感（两个Windows界面），这次Win8.1 2014 Update下，桌面用户可以在开始屏幕直接对Modern应用/桌面Win32程序采取右键式的管理，找回“桌面”的右键体验。

7、该更新将预装到新PC设备

在完成Windows 8.1 2014 Update RTM版本后，微软将把最新的Win8.1版本分发到其合作的OEM厂商，用于预装PC设备或者平板电脑电脑设备，即在2014年中期，市面上的新电脑设备将预装最新的Win8.1 2014 Update系统。
